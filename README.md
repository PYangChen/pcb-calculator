# PCB Design Calculator

> 單檔靜態網頁，無需安裝、無需網路，直接以瀏覽器開啟即可使用的 PCB 設計計算工具。

🔗 **線上使用**：[https://PYangChen.github.io/pcb-calculator/](https://PYangChen.github.io/pcb-calculator/)

---

## 功能總覽（30 個計算器）

### 走線寬度 / 電流
| # | 計算器 | 用途 |
|---|--------|------|
| 1 | IPC-2221 走線寬度 | 依電流、溫升、銅厚算出最小走線寬度 |
| 2 | 走線最大電流 | 已知尺寸，反推最大安全電流 |
| 3 | 走線電阻 / 電壓降 | 計算走線電阻、電壓降、功率損耗 |

### 阻抗控制
| # | 計算器 | 用途 |
|---|--------|------|
| 4 | Microstrip Z0 | 表層走線特性阻抗（常見目標 50 Ω） |
| 5 | Stripline Z0 | 內層走線特性阻抗 |
| 6 | 差分 Microstrip | 表層差分對阻抗（目標 90/100 Ω） |
| 7 | 差分 Stripline | 內層差分對阻抗 |
| 8 | CPWG 阻抗 | 共面波導接地 RF 結構阻抗 |

### 過孔 (Via)
| # | 計算器 | 用途 |
|---|--------|------|
| 9 | Via 電流容量 | 通孔最大安全電流（IPC-2221） |
| 10 | Via 電阻 | 通孔直流電阻 |
| 11 | Via 電感 | 通孔寄生電感及高頻感抗 |
| 12 | Via 寄生電容 | Pad 與 Antipad 間同軸電容 |
| 13 | Via Stub 共振 | Stub 四分之一波長共振頻率 |
| 14 | 環寬檢查 | Annular Ring 製程符合性驗證 |
| 15 | 板厚孔徑比 | 通孔電鍍製程難度評估 |

### 訊號完整性
| # | 計算器 | 用途 |
|---|--------|------|
| 16 | 趨膚深度 | 高頻電流集中深度與 AC/DC 電阻比 |
| 17 | 臨界走線長度 | 判斷走線是否需要按傳輸線設計 |
| 18 | 傳播延遲 | 訊號在走線上的傳播時間 |
| 19 | 反射係數 / RL | 阻抗不匹配造成的反射 |
| 20 | 串擾估算 | 相鄰走線串擾量，驗證 3W/5W 規則 |
| 21 | 微帶線串擾電壓 | 直接計算串擾電壓幅度 |
| 22 | 帶狀線串擾電壓 | 支援對稱/非對稱帶狀線串擾電壓 |
| 23 | 等長計算 | 時序偏差換算走線長度補償量 |
| 24 | 介質波長 | 特定頻率下 PCB 介質中的波長 |
| 25 | 走線自感 | 孤立走線本身的寄生電感 |
| 26 | 回路電感 | 走線＋地平面回流路徑的完整回路電感 |

### BGA 封裝
| # | 計算器 | 用途 |
|---|--------|------|
| 27 | BGA Via 尺寸建議 | 對角間隙限制下的 Via Pad 最大直徑及 Dog-bone 走線長度 |
| 28 | BGA 走線最小寬度 | n 條走線在兩 pad 之間所需的最小走線寬度與間距 |

### 電源完整性
| # | 計算器 | 用途 |
|---|--------|------|
| 29 | 旁路電容 SRF | 去耦電容有效頻率上限（自諧振頻率） |
| 30 | PDN 目標阻抗 | 電源分配網路最大阻抗目標值 |

## 使用方式

1. 直接開啟 `index.html`（Chrome / Firefox / Edge / Safari），無需安裝或網路連線
2. 點選上方分頁切換計算類別
3. 填入參數後按「計算」顯示結果；再按一次收合
4. 右上角可在 **mm** 與 **mil** 之間切換，所有輸入值自動換算

## 技術規格

- 純靜態單檔（`index.html`）：HTML + CSS + JavaScript

- 支援離線使用

---

## 公式來源

- 走線電流：IPC-2221 Table 6-1
- Microstrip 阻抗：Hammerstad-Jensen 經驗公式
- Stripline 阻抗：Wadell / IPC-2141
- CPWG 阻抗：Ghione-Naldi 共形映射法
- Via 電感：Goldfarb-Pucel 模型
- Via 電容：同軸電容模型
- 走線自感：Grover 公式
- 回路電感：部分電感法

---

## 注意事項

本工具提供設計估算值，實際製板前請以 PCB EDA 工具的 **Field Solver** 進行精確阻抗仿真，並與板廠確認疊構。
